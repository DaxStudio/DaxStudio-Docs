# DAX Studio version 3.2.0

Today we are happy to announce the release of version 3.1.0 of DAX Studio which includes the following new features and fixes.

## Updates / New Features
* Improving [Smart Paste](https://daxstudio.org/docs/features/smart-paste/)
* <Issue id="1288" /> Added Support for [View Metrics StatsColumnBatchSize](https://daxstudio.org/docs/features/model-metrics/#metric-options-dialog)
* Added sorting to ViewAs roles
* Fixed styling so trace command bars are not blurred

## Fixes
* Fix caturing server timing events when Auto ClearCache is active against powerbi:// endpoints
* Fix <Issue id="1284"/> dscmd CSV command not working with --connectionstring option
* Fix <Issue id="1283"/> dscmd export command --tables option parsing
* Fix <Issue id="1282"/> QueryBuilder measure renaming not working
* Fix <Issue id="1289"/> dscmd CSV command error when exporting some queries
* Fix sizing of status bar server name tooltip
